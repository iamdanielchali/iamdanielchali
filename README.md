# Daniel Chali • Tech Portfolio\n\nA clean, responsive personal portfolio built with HTML, CSS, and JavaScript. It showcases projects, experience, education, and skills, and is deployed via GitHub Pages.\n\n## Live Demo\n\nVisit: https://iamdanielchali.github.io/iamdanielchali/\n\n## Overview\n\nThis repository contains my portfolio website. It is lightweight, accessible, and easy to run locally. The site includes sections for Projects, Experience, Education, Skills, and Contact, with a dark/light theme toggle.\n\n## Features\n\n- Modern, responsive UI with Inter font and theme toggle\n- Hero section with avatar, headline, and call-to-action buttons\n- Featured Projects:\n  - CampusCare: Smart reporting platform to track student maintenance and discipline cases across campus (MERN, Analytics, Accessibility)\n  - Smart Attendance: QR-based classroom attendance tracking with analytics (PWA, Firebase, Charts)\n  - Learning Tools Suite: Quiz generator, flashcards, and note organizer (Vanilla JS, IndexedDB, Accessibility)\n- Social links enhanced with Font Awesome icons: GitHub, LinkedIn, Facebook, TikTok, Instagram, YouTube, Email\n- Skills highlighting frontend, backend, and tools:\n  - Python, MERN (MongoDB, Express, React, Node.js), React Native (mobile), AI/ML, Robotics\n  - Graphic Design (see designs: https://github.com/iamdanielchali/gown-poll)\n\n## Tech Stack\n\n- HTML5, CSS3, JavaScript (ES6+)\n- Fonts: Google Fonts (Inter)\n- Icons: Font Awesome\n\n## Getting Started\n\nNo special prerequisites are required.\n\n1. Clone the repository:\n\n```bash\ngit clone https://github.com/iamdanielchali/iamdanielchali.git\ncd iamdanielchali\n```\n\n2. Open `index.html` directly in your browser, or use a simple local server:\n\n- VS Code: Install the Live Server extension and click "Go Live"\n- Python (optional): `python -m http.server 5500` and open http://localhost:5500/\n\n## Deployment (GitHub Pages)\n\nThis site is deployed via GitHub Pages. To deploy or re-deploy:\n\n1. Push changes to the `main` branch\n2. In GitHub: Settings → Pages → Deploy from Branch\n3. Branch: `main`, Folder: `/ (root)`\n4. Wait for the build to complete; your site will be available at:\n\nhttps://iamdanielchali.github.io/iamdanielchali/\n\nNote: `.nojekyll` is included to ensure assets are served correctly without Jekyll processing.\n\n## Project Structure\n\n```text\n.\n├── index.html\n├── styles.css\n├── script.js\n├── .gitattributes\n└── .nojekyll\n```\n\n## Contact\n\n- Email: iamdanielchali@gmail.com\n- GitHub: https://github.com/iamdanielchali\n- LinkedIn: https://www.linkedin.com/in/danielchali\n\nFeel free to open issues or reach out for collaboration.
